# safety-app-workers
Application de prévention et de contrôle des risques de sécurité majeurs sur un nouveau poste de travail
Application mobile destinée à informer les travailleurs (intérimaires, jeunes, précaires) des risques de leur poste et des règles essentielles de sécurité.

## Contenu du dépôt
- frontend : application mobile (React Native)
- backend : API Node.js
- data : fiches risques par métier

## Objectif
Réduire les accidents du travail en donnant une information simple, rapide et accessible dès le début de la journée.
